Sockets

CPUs connect to the motherboard via their sockets.

Typically, Intel's sockets have pins that touch their CPUs. Sockets prior to LGA 775 (2004), however, are not like this.

Intel Sockets

2015

LGA 1151

Compatible with "Skylake" processors.
Compatible with chipsets H110, H170, Z170, B150, Q150, and Q170.
Anticipated to be relased in August of 2015.
Type: Desktop

2014

LGA 2011v3

Compatible with "Haswell-E, EP, EX" processors.
Compatible with chipset X99.
Type: Server/Workstation

2013

LGA 1150 (H3)

Compatible with "Haswell(-WS)", "Haswell Refresh", and "Broadwell" processors.
Compatible with chipsets H81, B85, Q85, Q87, H87 and Z87 (Haswell) as well as H97 and Z97 (Haswell Refresh, Broadwell).
Type: Desktop

2011

LGA 2011

Compatible with "Sandy Bridge-E, EP" and "Ivy Bridge-E, EP" processors.
Compatible with chipset X79.
Type: Server/Workstation

LGA 1155 (H2)

Compatible with "Sandy Bridge" and "Ivy Bridge" processors.
Compatible with chipsets B65, H61, Q67, H67, P67, and Z68 (Sandy Bridge) as well as B75, Q75, Q77, H77, Z75, and Z77 (Ivy Bridge).
Type: Desktop

2010

LGA 1567 (LS)

Compatible with "Beckton/Nehalem-EX" and "Westmere-EX" processors.
Chipset compatibility uncertain.
Type: Server

LGA 1248

Compatible with "Tukwila" and "Poulson" Itanium 9300/9500 processors.
Chipset compatibility uncertain.
Type: Server/Supercomputer

2009

LGA 1156 (H)

Compatible with "Lynnfield" and "Clarkdale" processors.
Compatible with chipsets H55, H57, P55, and Q57, as well as server chipsets 3400, 3420, and 3450.
Type: Desktop

2008

LGA 1366 (B)

Compatible with "Gulftown" and "Bloomfield" processors.
Compatible with chipset X58.
Type: Server/Workstation

2006

LGA 771 (J)

Compatible with "Clovertown", Woodcrest", "Wolfdale-DP/CL", "Harpertown", "Dempsey" and "Yorkfield-XE/CL" processors.
Chipset compatibility uncertain.
Type: Server

2004

LGA 775 (T)

Compatible with "Wolfdale(-3M)", Yorkfield", "Kentsfield(-XE)", "Bearlake", Conroe(-XE/L)", "Allendale", "Prescott", and "Cedar Mill" processors.
Compatible with chipsets i845GV, GE, i848P, i865G, GV, P, PE, i910GL, i915G, GL, GV, P, PL, i925X, XE, i945, 955, i945G, P, i955X, i946, 946GZ, PL, 965, i975, Q965, P965, G965, Q963, i975X, X35, P35, Q35, G35, P33, G33, Q33, P31, G31, X38, X48, P45, P43, G45, G43, G41, B43, Q43, and Q45.
Type: Desktop

2002

Socket 604

Compatible with "Gallatin", "Nocona", "Cranford", "Potomac", "Irwindale", "Tulsa", "Tigerton", "Dunnington", and "Paxville DP/MP" processors as well as some "Foster" and "Prestonia" processors.
Type: Server

PAC611

Compatible with "McKinley", "Madison(9M)", "Deerfield", "Fanwood", and "Hondo MCM" Itanium 2 processors.
Type: Server/Supercomputer

2001

Socket 603

Compatible with "Foster(MP)" and some "Prestonia" processors.
Type: Server

PAC418

Compatible with "Merced" Itanium processors.
Type: Server/Supercomputer
